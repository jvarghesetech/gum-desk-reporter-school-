# 🧹 Gum Desk Reporter

**Teachers report gum stuck under classroom desks so custodians can quickly locate and clean them.**

A simple, dark-themed web app for schools to track and manage gum removal requests. Teachers submit reports, admins review them, and custodians get printable work orders.

## 🌐 Live Demo

👉 **[Try it here](https://jvarghesetech.github.io/gum-desk-reporter-school-/)**

---

## ✨ Features

- **📝 Submit Reports** — Teachers report gum with room number and optional desk location
- **🔐 Admin Login** — Secure dashboard access for staff (username: `admin` / password: `admin1*`)
- **📊 Dashboard** — View all reports, mark as cleaned, track pending vs resolved
- **🖨️ Print View** — Generate custodian work orders grouped by room
- **💾 Local Storage** — Reports persist in browser, no server needed
- **📱 Responsive** — Works on desktop, tablet, and mobile

---

## 🚀 Quick Start

1. Download `index.html`
2. Open in any browser (Safari, Chrome, Firefox)
3. Start reporting!

No installation, no server, no dependencies.

---

## 📋 How It Works

### For Teachers
1. Click **New Report**
2. Enter the **Room** (required)
3. Enter your **Name** (required)
4. Optionally specify the **Desk Number**
5. Submit!

### For Admins
1. Click **Admin Login**
2. Enter credentials (`admin` / `admin1*`)
3. View all reports on the **Dashboard**
4. Mark reports as **Cleaned** when done
5. Use **Print View** to generate work orders for custodians
